# 🔍 机构真伪查询助手

> 官方查询入口汇总，让骗子无处遁形

## 📋 简介

这是一个帮助普通民众查询社会组织、企业、协会等机构真伪的工具网站。整合了官方查询入口，降低查询门槛，防范诈骗。

## 🎯 解决的问题

- 不知道去哪查询机构真假
- 官方平台分散，找不到入口
- 查询步骤复杂，不会操作
- 3.15 曝光的假机构套路识别

## 🔗 主要功能

### 社会组织查询
- 全国社会组织查询（民政部）
- 涉嫌非法社会组织名单
- 行业协会商会收费公示

### 企业/机构查询  
- 国家企业信用信息公示系统
- 信用中国
- 高频失信企业名单

### 特定行业查询
- 全国医疗卫生机构查询
- 养老服务机构备案查询
- 家政服务机构查询

### 防骗指南
- 骗子特征识别
- 举报渠道汇总
- 常见问题解答
